Exercício de JavaScript

1.

Crie um programa que solicite ao usuário seu nome e, em seguida, exiba no console:
“Olá, [nome]! Seja bem-vindo(a).”

Resposta 
const nome = prompt("Digite seu nome:");
console.log(`Olá, ${nome}! Seja bem-vindo(a).`);

2.

Solicite dois números e exiba a soma no console.
Dica: use Number() ou parseInt().

Resposta 
const num1 = prompt("Digite um número:");
const num2 = prompt("Digite outro número:");
const soma = parseInt(num1) + parseInt(num2);
alert(`A soma dos números é: ${soma}`);

4.

Peça um número e exiba o dobro no console.
“Olá, o dobro do número é.”

Resposta 
const num = prompt("Digite um número:");
const doub = parseInt(num) * 2;
alert(`O dobro do número é: ${doub}`);

5.

Solicite base e altura e exiba a área do retângulo.
Fórmula: área = base * altura

Resposta 
const base = prompt("Digite a base:");
const altura = prompt("Digite a altura:");
const area = parseFloat(base) * parseFloat(altura);
console.log(`A área do retângulo é: ${area}`);

6.

Peça o preço e o desconto e exiba o valor com desconto.
Exemplo: R$ 100, 10% → R$ 90

Resposta 
const preco = prompt("Digite o preço:");
const desconto = prompt("Digite o desconto (%):");
const precoDesconto = parseFloat(preco) - (parseFloat(preco) * parseFloat(desconto) / 100);
console.log(`O preço com desconto é: ${precoDesconto}`);

7.

Peça nome e sobrenome e exiba o nome completo.
Ex: João Silva

Resposta 
const nome = prompt("Digite seu nome:");
const sobrenome = prompt("Digite seu sobrenome:");
console.log(`Seu nome completo é: ${nome} ${sobrenome}`);

8.

Use confirm() para perguntar se o usuário quer se cadastrar.
Se "OK", peça nome e idade com prompt().
Caso "Cancelar", exiba "Cadastro cancelado".

Resposta 
const desejaCadastrar = confirm("Você deseja se cadastrar no site?");
if (desejaCadastrar) {
  const nome = prompt("Digite seu nome completo:");
  const idade = prompt("Digite sua idade:");
  console.log("Cadastro realizado com sucesso!");
  console.log("Nome: " + nome);
  console.log("Idade: " + idade);
} else {
  console.log("Cadastro cancelado pelo usuário.");
}
